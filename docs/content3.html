Microsoft Edge

Is it any better than IE?

How does Edge stand up to the competition?

One of Microsoft’s big changes with Windows 10 is that is came pre-installed with Microsoft Edge instead of Internet explorer. This was a good sign for the future of Microsoft’s web browsers as it finally released an up to date web browser. Microsoft edge seems to be on-par with the specs of the current big browser, Firefox and Chrome and hopefully it will completely replace Internet Explorer.

Microsoft Edge is the first major rebrand of Microsoft’s browser in over two decades. The browser was a big step for Microsoft as it has begun to move away for Internet Explorer and now use a modern browser.

Microsoft added a much friendlier user interface as well as huge improvements in running speed and extension support. It also released on IOS and Android and can sync features between your phone and desktop.

The design of Edge is much more modern and cleaner. Thicker squared off tabs, a 2D design, and a simple monochrome color palette for the window make the new browser look much sleeker.

While all of these bring it up to date, it just does not stand apart from the rest of the browsers. Edge is not the fastest, it is not the most chock full of features, it is not the most widely used, it is just another browser. The only thing going for it is that it’s preinstalled in all Windows computers. The problem is most people use it to straight away download another browser.

Will all this said, the hardest part for Edge to become accepted is that it is the successor to Internet Explorer. The stigma behind IE and the troubles it has caused everyone from web developers to the most basic users severely impedes people ability to accept Edge as a browser. Unless Edge can stand apart from the other browsers, it might have a hard time being accepted.
